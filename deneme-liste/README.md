
# Liste İşlemleri
Bu proje, kullanıcıdan ürün adı ve fiyat bilgisi alarak bir liste oluşturur ve bu liste üzerinde çeşitli matematiksel işlemler (toplam, medyan, ortalama) yapar.


## Nasıl çalıştırılır.
```bash
python app.py

```

## Örnek Çıktı

Hoşgeldiniz..


        0. Ürün Ekleyin.
        1. Listeyi Görün.
        2. Ürünlerin toplamını hesapla.
        3. Ürünlerin medyanını hesapla.
        4. Ürünlerin ortalamasını hesapla.
        5. Çıkış

Yapmak istediğiniz işlemi seçiniz: (0/5)0
Ürünün ismini yazınız lütfen: (Çıkış yapmak için q'ya basınız.)pasta
Ürünün tutarını giriniz lütfen: 40
[{'ad': 'pasta', 'tutar': 40}]
Ürünün ismini yazınız lütfen: (Çıkış yapmak için q'ya basınız.)kola
Ürünün tutarını giriniz lütfen: 20
[{'ad': 'pasta', 'tutar': 40}, {'ad': 'kola', 'tutar': 20}]
Ürünün ismini yazınız lütfen: (Çıkış yapmak için q'ya basınız.)q
