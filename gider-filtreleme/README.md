
# Gider Filtreleme Uygulaması
Bu uygulama ile giriş yapmış olduğunuz giderlernizi tarihlerine ve etiketlerine göre filtreleme yapabiliniz. 
Uygulama eğer yanlış bir format ile giriş yapıldığını tespit ederse hata gösterir.

## Nasıl çalıştırılır?

```bash
python main.py

```

## Örnek Çıktı

Hangi tarih için işlem yapmak istiyorsunuz: (gün, ay, yıl şeklinde giriniz.)30.11.2025
Hangi etiket için işlem yapmak istiyorsunuz: (yemek, market, eglence)yemek
Sonuç: []
Sonuç: [{'tarih': '28.11.2025', 'etiket': 'yemek'}]
